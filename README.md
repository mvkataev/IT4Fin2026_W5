# Техническое задание для дополнительного проекта

# Контекст
Необходимо проверить, добавляют ли текстовые данные предсказательную/объясняющую силу финансовым моделям. Для этого необходимо сконструировать NLP-признаки, объединить их с рыночными и фундаментальными данными и провести ablation study — сравнение моделей «с текстом» и «без текста».

# Цель исследования
Определить, помогают ли тексты предсказывать / объяснять доходность акции компании.


# 1. Сбор текстовых данных.
Сбор данных трёх типов на единой временной сетке по недельному разрезу:
рыночные (MOEX API): open, close, volume;
фундаментальные (SmartLab): P/E, ROE, ROE;
NLP-признаки (новости с сайта SmartLab): новости по компании.
Ожидаемый результат: Собранный и очищенный датасет с временными метками, код сбора приложен и воспроизводим.

# 2. Конструирование NLP-признаков
Создание признаков из трёх методов и агрегирование их по временным окнам.
Ожидаемый результат: Таблица NLP-признаков (признак → метод-источник → формула агрегации → смысл), признаки привязаны к временной сетке без look-ahead bias.
Три основных метода:
BERT-classification (avg_sentiment_7d, sentiment_std_7d, pct_negative_7d, sentiment_change);
NER, Named Entity Recognition (mentions_count_7d, unique_sources, co_mention_count);
Topic Modeling (topic_risk_share, topic_entropy, dominant_topic_id).

Возможно рассмотрение признаков взаимодействия между методами (например, sentiment × topic_risk); признаки на разных окнах агрегации (7 / 14 / 30 дней) с обоснованием выбора.

# 3. Дизайн исследования
Проектирование корректной схемы эксперимента: целевая переменная, разбиение выборки, схема ablation.

Обязательные элементы:
Есть целевая переменная по акции компании (бинарное направление доходности и превышение порога) с обоснованием.
Разбиение train/test по времени (не случайное) с контролем look-ahead bias.
Схема ablation (минимум 3 ступени): Baseline → +NLP → All / Only NLP.
Возможно рассмотрение Walk-forward / time-series cross-validation и нескольких целевых переменных (сравнение горизонтов прогноза).

# 4. Модель и оценка
Построение предсказательных и/или объясняющих моделей и их корректная оценка в рамках ablation.
Ожидаемый результат: Несколько моделей, таблица результатов «конфигурация признаков × модель × метрика», несколько метрик качества..
Возможно сравнение интерпретируемой (логистическая/линейная регрессия) и «чёрного ящика» (Random Forest / Gradient Boosting) моделей, анализ важности признаков (feature_importance / коэффициенты).

# 5. Финансовые метрики и итоговая интерпретация результатов
Ответ на главный вопрос исследования: помогают ли тексты предсказывать доходность акции?
Ожидаемый результат: Содержательные выводы. Должно быть раскрыто, дал ли NLP прирост в качестве предсказания. 

Также возможно отражение следующих аспектов:

•  Эконометрика: формальная проверка вклада NLP через вложенные модели (F-тест / Likelihood Ratio test), проверка мультиколлинеарности (VIF), обсуждение эндогенности. 
•  Машинное обучение: SHAP-анализ, корректная walk-forward валидация, работа с дисбалансом классов.

Необходимо превратить предсказания модели в финансовый результат, то есть выйти за рамки оценки точности и оценить экономическую ценность текстового сигнала (знания портфельного инвестирования):

• построить торговую стратегию на сигналах модели и оценить её финансовыми метриками (Sharpe ratio, MDD, кумулятивная доходность vs buy-and-hold);
•  учесть транзакционные издержки;
• сравнить ablation-портфели: даёт ли NLP-сигнал прибавку к Sharpe, а не только к F1/ROC-AUC;
• содержательный вывод: улучшение метрики не означает улучшение по деньгам.
